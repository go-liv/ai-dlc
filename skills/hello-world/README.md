# Hello World

**Type:** Skill  

## Description
Example skill template. Use as a starting point when creating new skills. Demonstrates proper structure and format.

## When to Use
- As a reference when creating a new skill
- To verify the DLC repo is accessible as context

## Procedure
1. Confirm Copilot can access this file via `Chat: Add Context` with the repo URL
2. Check that this text loads when referenced
3. Use this as a template to create your own skill

## Skill File Layout
```
skills/<your-skill>/
├── README.md          # Required — instructions and metadata
├── scripts/           # Optional — executable scripts
├── references/        # Optional — supplementary docs
└── assets/            # Optional — templates, boilerplate
```
